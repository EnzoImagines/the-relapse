# Dream Subway

An endless underground transit system that exists in the space between sleep and waking.

## Experience

- **Stations for places you've never been but somehow recognize**
- **Trains that arrive but you're never sure if the doors will open**
- **Maps on walls that change when you look away**
- **Fluorescent lights that flicker and die in patterns that feel meaningful**
- **Platform announcements for destinations that don't exist**

Built with Three.js. No frameworks. Just geometry and the uncanny.

## Controls

- **WASD**: Move around the platform
- **Mouse**: Look around
- **Click**: Lock mouse cursor
- **ESC**: Exit pointer lock

## Features

- Procedural station generation with liminal lighting
- Dynamic train arrivals and departures
- Flickering fluorescent fixtures (some dead, some dying)
- Ambient audio (60Hz electrical hum)
- Station signs that change when you're not paying attention
- Empty benches and trash cans that feel like they're waiting for someone

## Running

Open `index.html` in a modern browser. Works best in a dark room.

## Inspiration

Those subway platforms in dreams where you're waiting for a train to somewhere you need to be, but you can't remember where that is or why you need to go there.

The way underground spaces feel after the last train has gone.

Memory consolidation and the architecture of almost-remembering.
